service目录结构：

service                           	                                
    ├─IAdministrativeAreaService                 	          // 行政区域
    ├─IConsultationService                                  // 会诊
    ├─IDrugService                                          // 药物
    ├─IExamService                                          // 检验
    ├─IFamilyRecordService                                  // 家庭档案
    ├─IHealthEventService                                   // 医疗服务活动
    ├─IHealthExaminationService                             // 体检
    ├─IHealthHistoryService                                 // 健康史
    ├─IInpatientDataService                                 // 住院信息
    ├─IModifyTraceService                                   // 修改记录
    ├─IOutpatientService                                    // 门诊
    ├─IPhysicalExamService                                  // 体检报告
    ├─IPopulaceService                                      // 人口统计
    ├─IReferralService                                      // 转诊
    ├─IStudyService                                         // 检查
    │               
    ├─basic                                                
    │  ├─IEHRNumberService                                 // 健康档案编号
    │  ├─IEhrSecurityService                               // 创建、查询、删除、更新用户
    │  ├─IOrganizationItemRelationService                  // 机构
    │  ├─IOrganizationService                              // 创建、查询、删除、更新机构
    │  ├─IPcbDicItemService                                // 字典服务
    │  └─IStandParameterCfgService
    ├─orgotherinfo 
    │  ├─IOrgEconomicRelationService                       // 机构与经济
    │  ├─IOrgEnvironmentRelationService                    // 机构与环境
    │  ├─IOrgGarbageRelationService                        // 垃圾处理
    │  ├─IOrgOldPeopleHomeRelationService                  // 机构养老院
    │  ├─IOrgSchoolRelationService                         // 学校
    │  └─IOrgWaterRelationService
    │  │                    
    ├─personal                     
    │  ├─IPersonalRecordManagmentService                   // 个人档案管理
    │  ├─IPersonalRecordService                            // 个人档案
    │  ├─IPersonCanceledService                            // 注销个人健康档案
    │  ├─IPlatformService                                  // 平台人员服务
    │  └─IStatisticReportService
    ├─star                                                  // 星级统计
    │  ├─IOneStarValueGetter                               // 标记一星统计
    │  ├─IRecordStarService                                // 星级获取
    │  ├─IStarValueGetter                                  // 计算星级
    │  ├─IThreeStarValueGetter			                  // 标记三星统计
    │  └─ITwoStarValueGetter                               // 标记2星统计
    └─statistics                                           
        ├─IRecordStatisticsService			              // 档案统计、星级统计
        └─IStatisticsService			                      // 人群、档案、社区分布统计等
